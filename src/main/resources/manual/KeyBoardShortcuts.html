Keyboard shortcuts

Ride tries to follow commonly used shortcuts when possible i.e. copy paste with ctrl + c, ctrl + v etc. also shortcut commands used in RStudio are many times the same in Ride.
ctrl + c copy selection
ctrl + shift + c toggle line comments
ctrl + enter Run the selected part or the current line if nothing is selected
ctrl + space Display suggestion based on keywords and base R functions
ctrl + f unidirectional search
ctrl + quote insert ^ (raised to the power of) i.e. same as **
ctrl + s save
tab indent the line or selected text with 2 spaces
shift + tab unindent the line or selected text with 2 spaces
ctrl + v paste copied content
ctrl + x cut selection
ctrl + y reverse undo (step forward)
ctrl + z undo (step back)
